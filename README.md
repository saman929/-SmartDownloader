Smart Downloader Pro
بارگیر هوشمند برای دریافت پرونده‌ها با استفاده از توان کارسازهای گیت‌هاب.

این ابزار به شما اجازه می‌دهد هر پرونده‌ای را از هر نشانی اینترنتی دانلود کنید، حتی اگر دسترسی مستقیم از شبکه شما دشوار باشد.
